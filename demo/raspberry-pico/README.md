# picoRTOS Raspberry Pico single core demo

## Description

This simple demo uses the Rpi Pico & tests the following:

  - ADC
  - GPIO
  - UART
  - I2C
  - PWM
  - SPI
  - Watchdog

## Requirements

Some wiring is required to perform this standalone test:

  - GP20 to GP6 (SDA)
  - GP21 to GP7 (SCL)
  - GP27 to GP5 (IPWM)
