# picoRTOS on GD32VF103 demo

## Description

This simple demo uses the GD32VF103 & tests the following:

  - /ADC/
  - GPIO
  - UART
  - /I2C/
  - PWM
  - SPI
  - Watchdog

## Requirements

Some wiring is required to perform this standalone test:

  - R0 to T0 (UART)
  - B6 to B10 (SCL)
  - B7 to B11 (SDA)
  - A6 to A7 (SPI)
