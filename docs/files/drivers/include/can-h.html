can_accept

int can_accept(struct can *ctx,
can_id_t id,
can_id_t accept_mask)

Registers a message for rx on a CAN interface

Parameters

ctxA CAN interface
idThe ID of the message to register
accept_maskThe acceptance mask for this ID

Remarks

0 as acceptance mask means only the specified ID will be accepted Other acceptance masks act as a logical AND between the received ID, the acceptance mask and the specified id

Returns

0 in case of success, -errno otherwise
