picoRTOS_SMP_set_core_mask

void picoRTOS_SMP_set_core_mask(picoRTOS_priority_t prio,
picoRTOS_mask_t core_mask)

Assign a task to specific core(s)

Parameters

prioThe task priority/identifier
core_maskThe cores to run the taks on in the form of a mask, where (1 << 0) is core 0, (1 << 1) is core 1, etc

Example

picoRTOS_add_task(&task, (picoRTOS_priority_t)TASK_PRIO);
picoRTOS_SMP_set_core_mask((picoRTOS_priority_t)TASK_PRIO, (picoRTOS_mask_t)(1 << 1));

Remarks

picoRTOS-SMP will throw a debug exception and stall if the mask is 0, doesn’t match the exact number of cores, if prio is > CONFIG_TASK_COUNT or is not assigned yet
